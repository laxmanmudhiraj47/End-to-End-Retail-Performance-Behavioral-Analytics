# 📌 Project Overview

This is an end-to-end data analytics project designed to transform raw retail transaction data into meaningful business insights. The project demonstrates the complete analytics workflow, including **data cleaning, exploratory data analysis (EDA), SQL-based business analysis, and interactive Power BI dashboards**.

The primary objective is to help retail businesses monitor sales performance, understand customer purchasing behavior, identify profitable products, and support data-driven decision-making.

---

# 🎯 Business Objectives

- Analyze overall retail sales performance
- Identify top-performing and low-performing products
- Understand customer purchasing behavior
- Compare regional and store-wise sales
- Analyze revenue and profitability
- Build interactive dashboards for business decision-making

---

# 📂 Raw Data

The project uses a comprehensive retail sales dataset containing transactional, customer, product, and store information.

### Dataset Contains

| Data Category | Description |
|---------------|-------------|
| 🧾 Order Information | Order ID, Order Date, Order Priority |
| 👤 Customer Information | Customer ID, Customer Name, Gender, Age, Customer Segment |
| 🛍️ Product Information | Product ID, Product Name, Category, Sub-Category |
| 🏬 Store Information | Store ID, Store Name, City, State, Region |
| 💰 Sales Information | Quantity, Unit Price, Discount, Sales, Cost, Profit |
| 🚚 Shipping Details | Ship Mode, Shipping Cost |
| 💳 Payment Details | Payment Mode |
| 🔄 Returns | Return Status |

### Dataset Summary

- Historical retail transaction records
- Customer demographics
- Product catalog
- Regional sales information
- Revenue and profit metrics
- Store performance data
- Shipping information
- Payment details
